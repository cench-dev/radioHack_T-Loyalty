# Loyalty Hub — единый раздел лояльности банка

Решение задачи: показать клиенту всю выгоду от банка в одном экране (кэшбэк, программы лояльности, акции от партнёров, прогноз, кросс-селл) с геймификацией и AI-советами.

## Стек

- Backend: FastAPI + SQLite + Pandas + scikit-learn + GigaChat API
- Frontend: Next.js 14  + TypeScript + Recharts (CSS Modules)
- Деплой: Railway / Render (бэк), Vercel (фронт)

## Что сделано

### Бэкенд
- Парсинг 5 CSV-файлов в SQLite при старте
- API из 11 эндпоинтов: пользователи, лояльность, программы, история, прогноз, AI-советы, офферы, кросс-селл, геймификация
- Прогноз кэшбэка через линейную регрессию на месячных агрегатах
- AI-советы от GigaChat API с rule-based фоллбеком (работает даже без ключа)
- Конверсия валют программ в рубли (рубли / мили / браво-баллы)
- Сегментация офферов и кросс-селла под `financial_segment` юзера
- Уровни лояльности (5 уровней по сумме кэшбэка)
- Ачивки и квесты (включая featured-квест "сохрани кэшбэк до следующего месяца")
- 11 pytest тест-кейсов

### Фронтенд
- 5 экранов: Логин, Дашборд, История, Партнёры, Достижения, Детали программы
- Light + Dark тема с автодетектом и кнопкой переключения
- Адаптивная вёрстка под десктоп + планшет + мобильный
- Поддержка увеличенных шрифтов (`prefers-reduced-motion` для анимаций)
- Логин без пароля: ввод email/телефона ИЛИ выбор из списка тестовых юзеров
- Обработка ошибок сети + skeleton-loaders
- Стиль T-Bank: жёлтый акцент `#FFDD2D`, скругления, минималистичные SVG-иконки

## Запуск локально

### Бэк
```bash
cd backend
python -m venv venv 
venv/Scripts/activate
pip install -r requirements.txt
cp .env.example .env
uvicorn app.main:app --reload --port 8000
```

### Фронт
```bash
cd web
cp .env.example .env.local
npm install
npm run dev
```

Открыть <http://localhost:3000>.
